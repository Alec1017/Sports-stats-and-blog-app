## Getting Started

1) Clone project

2) Navigate into application

```bash
cd sports-stats-and-blog-app
```

3) Install using yarn (this project uses node v14.16.0)

```bash
yarn install
```

4) Run development server

```bash
yarn run dev
```

The app will be running on http://localhost:3000 by default
